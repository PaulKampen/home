Свиные ребрышки

Основные характеристики

Альтернативное название: свиные ребра-гриль
Вид кухни: интернациональная
Событийность: пикник
Порции: 1
Время: 45
Категория: из мяса и птицы

Тип блюда

по основному продукту: мясо
по способу приготовления: запекание
по подаче: закуски

Приготовление

по сложности: средне
по времени: средне

Ингредиенты

Ребра свиные 300 г
Соль по вкусу
Перец по вкусу
Чеснок по вкусу
Лук по вкусу
Соевый соус по вкусу
Способ приготовления
Свиные ребра промываем и обсушиваем. Натираем ребра солью, перцем, давленым чесноком. Добавляем нарезанный лук и соевый соус. Маринуем 30мин. После обжариваем ребрышки на мангале до румяной корочки.
Подача и сервировка кальмаров с майонезом
На гарнир к ребрышкам лучше подать овощной салат.
