Первый раз.

<ol>
	<li>Установить git (Git Bush для Windows)</li>
	<li>Зайти в терминал (Git Bush)</li>
	<li>Склонировать репозиторий командой - `git clone https://github.com/nick-levelup/home.git`</li>
	<li>Проверить склонированный репозиторий (`C:\Users\{USERNAME}\home`)</li>
</ol>
-----
<ol>
	<li>После выполнения ДЗ. Студент создает папку в директории со своим именем с номером занятия.</li>
	<li>Туда копирует выполненное задание</li>
	<li>Выполнить команды: (следует выполнять с home директории, в Git Bush должно быть приблизительно так - `{ИмяПользователя} ~/home (master)`)</li>
	<li>`git pull` (затянуть изменения С сервера)</li>
	<li>`git add .` (добавляет файл в индекс)</li>
	<li>`git commit -am 'ДЗ {имя_студента}'` (коммит изменений)</li>
	<li>`git config --global {user.email} "{your@email}"` (опциально, если ошибка при создании коммита)</li>
	<li>`git pull` (затянуть изменения С сервера)</li>
	<li>`git push` (затянуть изменения НА сервер) (необходим логин пароль + добавлен ваш аккаунт)</li>
</ol>
